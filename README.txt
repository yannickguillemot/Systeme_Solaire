Projet d'Introduction au multimédia. 
Réalisé par Guillemot Yannick et Grosperrin Quentin
Printemps 2011

Pour lancer le projet : 
Lancer application.exe (Windows), application.app (Mac), application.swf dans /publications

Il semble que l'application soit plus fluide en utilisant le fichier .swf.
Si malgré cela, vous rencontrez des problèmes de fluidité : tentez de relancer l'application ou de redémarrer votre ordinateur.


Sources disponibles dans le dossier du même nom.

Le rapport de ce projet est consultable via le fichier .pdf fourni.
